Running the Gridlock app (dev & production)

Prerequisites

- Node.js 20.19+ recommended (Vite warns below 20.19). Python 3 for data processing scripts.

Dev (start API + Vite)

```bash
npm install
npm run dev
```

Build (production)

```bash
npm run build
npm run preview
```

Live stream

- The app is always connected to the live stream built from the last 25 events in `src/data/real/demo_stream.json`.
- Incidents advance every 2 minutes through the shared store in `src/stores/liveStreamStore.ts`.
- Global alerts appear in the top-right and are generated by `src/components/shared/LiveSupervisor.tsx`.
- The status bar now shows the current incident and live window count.

Data pipeline

- Regenerate processed artifacts with:

```bash
python scripts/process_data.py
```

Notes

- To add live behavior to additional modules: import `useLiveFeed()` and render `feed.currentIncident` or `feed.currentEvents` directly.
